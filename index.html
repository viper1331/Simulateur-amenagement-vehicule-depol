Modules

Sélectionnez un module prêt à l'emploi, importez une bibliothèque d'objets 3D ou créez vos propres équipements pour composer votre aménagement.

Les bibliothèques peuvent être fournies au format JSON ou sous forme d'archives ZIP (fichiers SH3F de Sweet Home 3D). Décompressez les archives si nécessaire : le simulateur détecte automatiquement le descriptif JSON et ajoute les modules au catalogue.

Sources de modèles 3D gratuits

  • Modèles 3D libres Sweet Home 3D — plus de 1 600 modules créés par la communauté.
  • Contributions 1.9.3 — 511 modèles (Licence Art Libre).
  • Scopia 1.9.3 — 500 modèles (Licence CC-BY).
  • Blend Swap CC-0 — 175 modèles (Domaine public).
  • Éditeur de bibliothèque Sweet Home 3D — créez vos propres fichiers SH3F.
  • Trimble 3D Warehouse — vaste catalogue Collada/KMZ (connexion Trimble requise).
  • Resources.blogscopia — modèles libres prêts à l'emploi.
  • Dodecagon — bibliothèques variées (domaine public).
  • Archive3D — nombreuses ressources OBJ/3DS.

Respectez les licences (Art Libre, CC-BY, CC-0, domaine public, etc.) mentionnées par les auteurs lors de l'import.
